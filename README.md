## Setup
Init virtual envirnment with ```python -m venv venv```
Launch virtual env with ```source ./venv/bin/activate```

Then, install dependencies with 
```
pip install -r pip install -r requirements.txt
```

## Run
Run with ```python 3 drone.py```
