## Setup
 ```
 python -m venv venv # Setup virtual environment (venv)
 source ./venv/bin/activate # Activate venv
 ```

Then, install dependencies with 
```
pip install -r requirements.txt
```

## Run
```
python3 drone.py # Run the project
```
